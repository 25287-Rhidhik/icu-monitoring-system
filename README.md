# ICU Monitoring System using Edge AI

## 📌 Project Overview
This project monitors ICU room conditions using sensors and Edge AI.  
It detects temperature, humidity, and predicts patient risk using a machine learning model.

## 🚀 Features
- Real-time temperature & humidity monitoring
- AI-based risk prediction
- Multi-patient dashboard
- Firebase integration
- Alert system (Critical / Warning / Safe)

## 🛠️ Technologies Used
- Python
- Streamlit
- Firebase
- TensorFlow Lite
- Arduino

## 📊 Output
- Live dashboard with patient data
- Graphs for temperature & humidity
- AI score prediction

## 🎯 Use Case
Helps hospitals monitor ICU conditions and detect abnormal situations early.
